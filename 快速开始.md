# ⚡ 快速开始 - 3 分钟上手

这是最简单的入门指南，让你快速玩上游戏！

---

## 🎯 我是完全的新手

### 第 1 步：安装 Node.js（只需一次）

1. 打开 https://nodejs.org/
2. 下载绿色的 "LTS" 版本
3. 双击安装，一路点 "Next"
4. ✅ 完成！

### 第 2 步：下载游戏

- 如果你已经有代码在电脑上，跳过这步
- 否则从 GitHub 下载或克隆代码

### 第 3 步：玩游戏！

**Windows 用户（最简单）：**

双击这个文件：**`启动游戏.bat`**

就这么简单！浏览器会自动打开游戏。

**其他系统或喜欢用命令行：**

```bash
npm install  # 第一次需要运行
npm run dev  # 启动游戏
```

然后打开浏览器访问 `http://localhost:5173`

---

## 🌐 我想和朋友联机玩

确保你们在同一个 WiFi 网络下。

### 方案一：超简单（Windows）

**房主（服务器）：**
1. 双击 **`启动联机服务器.bat`**
2. 记下显示的 IP 地址（如 192.168.1.100）
3. 双击 **`启动游戏.bat`**

**朋友：**
1. 双击 **`启动游戏.bat`**
2. 在游戏中点"局域网对战"
3. 输入房主的 IP：`http://192.168.1.100:3001`
4. 加入房间就能玩了！

### 方案二：命令行

**房主：**
```bash
# 终端 1
npm run server

# 终端 2
npm run dev
```

**朋友：**
```bash
npm run dev
```

详细步骤看：[BEGINNER_GUIDE.md](./BEGINNER_GUIDE.md) 第二部分

## 🚀 我想发布到网上让所有人玩

### 超快速版本（5 分钟）

1. **注册 GitHub**（如果没有）
   - 访问 https://github.com
   - 点击 Sign up

2. **创建仓库**
   - 点击 "+" → "New repository"
   - 名字必须是：`cabin_game`
   - 选择 Public
   - 创建

3. **上传代码**（在游戏文件夹打开命令提示符）
   ```bash
   git init
   git add .
   git commit -m "初始提交"
   git branch -M main
   git remote add origin https://github.com/你的用户名/cabin_game.git
   git push -u origin main
   ```

4. **启用 GitHub Pages**
   - GitHub 仓库 → Settings → Pages
   - Source 选择：**GitHub Actions**

5. **等待 3 分钟**
   - 访问：`https://你的用户名.github.io/cabin_game/`
   - ✅ 完成！

详细图文教程看：[DEPLOY_TO_WEB.md](./DEPLOY_TO_WEB.md)

## ❓ 遇到问题？

### 常见错误速查

**"npm 不是内部或外部命令"**
→ 没装 Node.js，去第 1 步

**"端口已被占用"**
→ 关闭之前运行的游戏（Ctrl+C）

**"无法连接服务器"**
→ 检查 IP 地址、WiFi 是否相同、防火墙

**"网页版打不开"**
→ 等待 5 分钟，检查 GitHub Actions 是否成功

更多问题看：[BEGINNER_GUIDE.md](./BEGINNER_GUIDE.md) 的故障排除部分
